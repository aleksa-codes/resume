    Next Meal

    React - Next.js - TypeScript - API - TailwindCSS - daisyUI

    Web application that generates a random meal recipe. The project demonstrates the use of Next.js and TypeScript. UI components are written in React and are styled using TailwindCSS together with daisyUI is used for themes and some predefined styles. The data for the recipes is fetched from TheMealDB API using a custom fetch hook that is implemented with the FetchAPI. The application is a PWA (Progressive Web App) and has Dark Mode.
